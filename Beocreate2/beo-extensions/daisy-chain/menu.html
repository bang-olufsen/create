Chaining

Chain Two Amplifiers

Connect and use two Beocreate 4-Channel Amplifiers together as a primary & secondary pair. Why and how.

This is an experimental feature.

Chaining was turned off, because the DSP program was upgraded. Learn more.

Chaining is enabled. Select preferred speaker roles in channel settings.

Why Chain Amplifiers?

If your speaker uses more than two amplifier outputs, playing in stereo is not possible with them using only one amplifier.

How It Works

The chaining feature allows you to connect two amplifiers with an optical audio cable so that they work together, each powering one speaker.

When chained, one of the speakers is the primary speaker that receives sound from your sources. The primary speaker will then relay the sound to the secondary speaker.

Chaining Turned Off By Itself

This may happen for any of the following reasons:

  • A different sound preset is selected
  • Levels or delays of individual speaker channels are changed
  • Speaker equaliser or crossover settings are changed
  • A DSP program is installed or upgraded

These settings can't be synchronised over the optical connection. To make sure both speakers behave and sound the same, set up the secondary amplifier and chaining again.

Set Up Chaining

Set up secondary amplifier

One amplifier needs to be set up to work without the Raspberry Pi. This will be used as the secondary amplifier.

If you have already set up the secondary amplifier, you can connect it now.

Settings stored

Next, shut down the product and unplug power before making connections.

Move Raspberry Pi

Remove the Raspberry Pi from the current amplifier and connect it to the other one. This will become the new primary amplifier.

Connect together

Connect an optical audio cable from the output of the primary amplifier (the one with Raspberry Pi) to the input of the secondary amplifier (the one without).

The output is covered by a white flap, the input has a removable cap.

Power up

Connect both speakers to power. The new primary amplifier will be automatically set up, which will take a moment.

Connect the other speaker to power.

The chain is then complete and you can select the preferred speaker roles in Channels.
